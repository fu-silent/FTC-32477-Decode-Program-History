## ⚙️ 基本配置

### 修改硬件名称

在 `Constants` 类或 `RobotConstants_1_0.java` 中：

```java
// 修改这些硬件名称
CHASSIS_MOTOR_FRONT_LEFT_NAME = "lf";      // 改成你的名称
CHASSIS_MOTOR_FRONT_RIGHT_NAME = "rf";
CHASSIS_MOTOR_BACK_LEFT_NAME = "lb";
CHASSIS_MOTOR_BACK_RIGHT_NAME = "rb";

SUBSYSTEM_INTAKE_MOTOR_NAME = "intake";
SUBSYSTEM_LOAD_MOTOR_NAME = "load";
SUBSYSTEM_SHOOTER_MOTOR_NAME_S1 = "s1";
SUBSYSTEM_SHOOTER_MOTOR_NAME_S2 = "s2";
```

### 修改常用参数

```java
CHASSIS_JOYSTICK_DEADZONE = 0.1;          // 死区（0-1，默认0.1）
TURN_SENSITIVITY_FACTOR = 0.8;            // 转向灵敏度（0-1）
SHOOTER_POWER_DEFAULT = 0.8;              // 默认发射功率（0-1）
SHOOTER_POWER_STEP = 0.05;                // 每次调节幅度
```

## 🎮 控制按键（v1.0 当前映射）

| 功能 | 按键 | 说明 |
|------|------|------|
| **前后移动** | 左摇杆 Y轴 | 向上前进、向下后退 |
| **左右平移** | 右摇杆 X轴 | 向右右移、向左左移 |
| **原地旋转** | 左摇杆 X轴 | 向右顺时针、向左逆时针 |
| **拾取进** | A 键 | 拾取马达正向运转 |
| **拾取退** | B 键 | 拾取马达反向运转 |
| **装填** | LT（左扳机） | 装填马达启动 |
| **发射** | RT（右扳机） | 发射马达启动 |
| **发射功率 +** | D-Pad ↑ | 功率增加 0.05 |
| **发射功率 -** | D-Pad ↓ | 功率减少 0.05 |
| **映射切换** | Y 键 | 切换线性/非线性映射 |

## ✨ v1.0 最新特性

✅ **发射功率控制**（替代转速 RPM）
- 范围：0.0 ~ 1.0
- 步长：0.05
- 调节：D-Pad 上/下

✅ **摇杆无输入立即刹停**
- 摇杆回中时车辆立即制动
- 采用 BRAKE 模式实现

✅ **正确的麦克纳姆轮控制**
- 修复了平移和旋转的符号错误
- 所有方向（前后左右旋转）都能正常工作
