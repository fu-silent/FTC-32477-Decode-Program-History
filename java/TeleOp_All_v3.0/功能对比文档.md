# v3.0 vs v2.3 功能对比

## 核心改进：精确导航与专业分工

| 方面 | v2.3 (基础版) | v3.0 (增强版) |
|---|---|---|
| **航向重置** | [cite_start]单点重置 (Back 键设当前为 0°)  | **多角度重置 (D-Pad)**：0°, ±90°, 180° |
| **自动转向** | [cite_start]单一自动转向 (RB 键)  | **双自动瞄准** (A/B 键)，目标角度可预设 |
| **驾驶分工** | 隐式分工 | **显式分工** (D1 全能，D2 瞄准/发射) |
| **瞄准控制** | D1 全速转向 | **D2 精确转向** (0.5x 减速) |
| **发射清理** | 无专用按键 | **D2 RB 键** 专用于 500 RPM 反转清理 |

---

## 详细对比

### 1. 导航系统：从校准到精确预设

**v2.3**：
- [cite_start]仅支持通过 Back/Options 键将机器人的当前朝向**校准**为新的 0 度，仅用于校正无头模式的参考系 。

**v3.0** ✨：
- **引入多角度重置**：驾驶员 1 可通过 D-Pad 直接将航向角重置到 0°、90°、180° 和 -90° 四个标准角度。
- 内部逻辑修复了 FTC SDK 中 `imu.resetYaw(target, unit)` 报错的问题，采用**软件偏移量**来实现非零角度重置，更稳定可靠。

### 2. 瞄准与发射：专业分工

**v2.3**：
- [cite_start]只提供一个自动转向功能 (RB 键)，通常用于一个预设角度 。

**v3.0**：
- **双自动瞄准**：D1 的 A 和 B 键分别对应两个预设的场向目标角度（例如 45° 和 61°），用于快速瞄准不同的目标区域。
- **D2 精确转向**：Driver 2 使用右摇杆接管转向时，底盘转向速度减半，实现更精细的瞄准调整。
- **发射反转专用键**：将 D1 的自动转向（RB 键）的功能改为 D2 的**发射器 500 RPM 反转**，用于在比赛中快速清理卡住的物品。

### 3. 共同保留的 v2.3 核心功能

- [cite_start]**无头模式** (Field-Centric) 与 **机器人模式** (Robot-Centric) 的 X 键切换功能得以保留，是 v3.0 的驱动基础 。
- [cite_start]**线性/非线性**驱动切换 (Y 键) 保留，用于适应不同驾驶习惯 。
